# Distance Pathway Work Area

This folder is separate from `mini_models` and is intended for the new distance-pathway development work.

Planned reports:

- `reports/simple_coincidence_model.md`: explanatory corollary-discharge and delay-line coincidence model.
- `reports/accuracy_optimisation_testing.md`: LIF, RF, and binary detector comparison.
- `reports/full_biological_model.md`: placeholder for the later fuller biological model.
- `reports/full_optimised_model.md`: placeholder for the later optimized model.

Run the current analysis with:

```bash
uv run --no-sync python distance_pathway/experiments/distance_pathway_analysis.py
```
